== Description ==

<p>This plugin extends WooCommerce and allows for payments to be made from credit/debit cards using Chase E-Xact payment gateway.</p>

<p>Tested up to WooCommerce version 5.0</p>

**Configuration**
1. Install and activate plugin
2. Navigate to WooCommerce > Settings > Payments
3. Ensure Chase E-xact Checkout is enabled
3. Click 'Manage' next to Chase E-xact Checkout
4. Fill out required information
5. Click 'Save changes'

**Title:** Controls the title presented to the customer during checkout.

**Description:** Controls the description presented to the user during checkout.

**API Login ID:** Provided by Chase E-xact. Required for verification purposes on their server.

**API Transaction Key:** Provided by Chase E-xact. Required for verification purposes on their server.

**API Mode:** Test mode for testing purposes only. Live mode for production.

**Transaction Success Message:** Message displayed to customer for a successful transaction.

**Transaction Failed Message:** Message displayed to customer for a failed transaction.
